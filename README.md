# Sniffer
IS301 Computer Communication and Network Project

计算机通信网大作业 - 网络抓包器

*作者 - Evander & Xynnn_*

## Usage

    pip3 install pyqt5
    pip3 install winpcapy
    python3 ./main.py
  
Note: Python 3.x only, and the script may contains any other packages, just search for installing them.
